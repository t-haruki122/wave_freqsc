# wavファイル 周期探索
## 概要
情報工学実験1 音・音声用のプログラムです．  
WinFdumpから出力されたtxtファイルを使用します．  

## 使い方
以下のコマンドを実行してください．
```
python3 main.py [ファイル名.txt]
```

## サンプルファイルについて
このリポジトリには，WinFdumpから出力されたサンプルファイルが内包されています．内容は以下の通りです．
- サンプリング周波数 : 8000 Hz
- ビット深度 : 16 bit
- 波形 : 正弦波 (200Hz)
- 時間 : 1.0 s  

サンプルファイルを解析するには，以下のコマンドを実行してください．
```
python3 main.py sample.txt
```
